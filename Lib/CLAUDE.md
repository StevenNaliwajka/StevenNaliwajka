# StevenNaliwajka/Lib

Reusable Python libraries.

## Architecture

Passthrough category node in the StevenNaliwajka org. Contains child repos in `repos/`. No project-specific code — this directory exists for organizational grouping.

## Child Repos

- Contact — Contact management library
- CSV — CSV file handling
- ENV — Environment variable management
- FileIO — File I/O utilities
- JSON — JSON handling
- MySQL — MySQL database library
- VENV — Virtual environment management

## Do Not

- Do not add code to this directory — code lives in child repos
- Do not edit wrapper scripts if present
