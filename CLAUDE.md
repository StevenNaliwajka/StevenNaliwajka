# StevenNaliwajka

Personal organization root. Contains 12 category groupings spanning games, hackathons, libraries, infrastructure, research, and tools.

## Architecture

Passthrough org node. Each subdirectory is a category that groups related repos under `repos/`. The tree is walked by the Configure system during lifecycle operations.

## Categories

| Category | Purpose |
|----------|---------|
| AI | AI integration projects |
| Game | Game development and modding |
| Hack | Hackathon projects |
| Infra | Infrastructure and networking |
| Lib | Reusable Python libraries |
| Personal | Personal projects |
| Print | 3D print quality tools |
| Research | Academic research |
| Security | Security tools and field work |
| Tool | Developer and system utilities |
| Web | Web development |
| Work | Work-related projects |
